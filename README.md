## Business Program 

## Installation
Use npm to install packages for Electron folder
```bash
cd Electron
npm i package
```
## Run Electron
```bash
cd Electron
npm start
```
## Run Website

# Mac
```bash
cd Typescript 
open index.html
```
# Linux
```bash
cd Typescript 
google-chrome index.html
```
or 
```bash
cd Typescript 
firefox index.html
```
# Window
```bash
cd Typescript 
start index.html
```
